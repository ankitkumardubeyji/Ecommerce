# How to setup the SpringBoot project
- Go to starter.spring.io
- Select the group bussiness unit ie : com.project
- Add the artifacts , name
- Add the dependencies
- Export

# Springweb
- for building rest apis
- for embedded server
- for spring mvc framework

# SpringDataJPA
- for the database interactions
- for the jpa implementations and functionalities

# H2 database
- inmemory database
- Data gets deleted from the database once the server stops

# Validation
- for annotations for validating the data ie getting saved in the database
